# 房地產謄本面積與租金計算器 (Property Area & Rent Calculator)

這是一個專為房地產從業人員、房東及投資客設計的網頁版計算器。能夠快速將建物謄本上的平方公尺數據，精準換算為坪數，並自動計算單坪租金與管理費效益。

本工具為純靜態網頁（HTML/CSS/JS），可直接部署於 GitHub Pages，無需後端伺服器，手機與電腦皆可使用。

## 🚀 功能特色

* **自動單位換算**：輸入平方公尺 ($m^2$)，自動換算為台灣常用坪數（係數 0.3025）。
* **持分精確計算**：針對公共設施與車位，支援輸入「總面積 × 分子 / 分母」，自動計算持分面積。
* **防呆機制**：
    * 主建物與附屬建物為**必填**，確保基礎數據正確。
    * 公設、車位、租金為**選填**，若留空系統自動視為 0。
* **響應式設計**：介面針對手機優化，適合外出看房或業務洽談時即時使用。

## 🧮 計算邏輯與公式

本計算器採用以下標準定義進行計算：

1.  **基本換算**：
    * `1 平方公尺 = 0.3025 坪`

2.  **面積定義**：
    * **總面積** = 主建物 + 附屬建物 + 公共設施持分 + 車位持分
    * **權狀面積** (計算租金用) = 總面積 - 車位面積
    * **室內面積** = 主建物 + 附屬建物

3.  **單價分析**：
    * **單坪月租金** = 月租金 ÷ **權狀面積坪數** (不含車位)
    * **單坪管理費** = 管理費 ÷ **總面積坪數** (含車位)

## 📖 使用說明

1.  打開網頁連結。
2.  依據建物謄本輸入 **主建物** 與 **附屬建物** 面積（必填）。
3.  若有公設或車位，輸入該項目的總面積以及持分比例（分子/分母）。
4.  輸入預期或實際的 **月租金** 與 **管理費**。
5.  點擊「計算結果」，即時查看詳細坪數分析與單價。

## 🛠️ 如何部署 (GitHub Pages)

本專案設計為單一檔案 (`index.html`) 即可運行，部署步驟如下：

1.  將本專案 Fork 或 Clone 到您的 GitHub 帳號。
2.  進入儲存庫的 **Settings** (設定)。
3.  在左側選單點擊 **Pages**。
4.  在 **Branch** 選項中，選擇 `main` (或 `master`) 分支，並點擊 **Save**。
5.  等待約 1 分鐘，重新整理頁面，上方將出現您的專屬網址。

## 📱 加入手機主畫面 (像 App 一樣使用)

* **iPhone (Safari)**：點擊下方分享按鈕 (方框向上箭頭) → 往下滑找到「加入主畫面」。
* **Android (Chrome)**：點擊右上角選單 (三點) → 選擇「加到主畫面」。
